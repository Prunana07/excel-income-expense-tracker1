# Excel Income & Expense Tracker

This project is a simple, beginner-friendly Excel template designed to help users track daily and monthly income and expenses.

## Features
- Daily transaction tracking
- Monthly structure
- Automated balance calculation
- Total and average income and expense summaries
- Category-based tracking
- Data validation to prevent incorrect entries
- Clean layout for non-Excel users

## Tools Used
- Microsoft Excel (Web and Desktop compatible)

## How to Use
1. Download the Excel (.xlsx) file
2. Open in Microsoft Excel or Excel on the web
3. Enter Date, Category, and Description
4. Enter either Income or Expense (not both)
5. Balances update automatically

## Use Cases
- Personal finance
- Small business bookkeeping
- Freelancers
- Students
